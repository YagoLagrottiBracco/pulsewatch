    **Imports:**
    - `useState`, `useRef`, `useCallback` from `react`
    - `Link` from `next/link`
    - `TrendingDown`, `AlertTriangle`, `Clock` from `lucide-react`
    - `Card`, `CardContent` from `@/components/ui/card`
    - `Input` from `@/components/ui/input`
    - `Button` from `@/components/ui/button`
    - `Badge` from `@/components/ui/badge`

    **State:**
    - `displayValue: string` (formatted input for display)
    - `revenue: number` (parsed numeric value, updated after 300ms debounce)
    - `timerRef: useRef<ReturnType<typeof setTimeout> | null>` (debounce timer — per D-05)

    **Formatting helper:**
    ```typescript
    const formatBRL = (value: number) =>
      new Intl.NumberFormat('pt-BR', { style: 'currency', currency: 'BRL' }).format(value)
    ```

    **handleChange callback (per D-04, D-05):**
    - Strip non-digits: `e.target.value.replace(/\D/g, '')`
    - Format display with thousand separator: `Number(digits).toLocaleString('pt-BR')`
    - Clear previous timer, set new 300ms timeout to update `revenue`

    **Computed values (per D-06, D-07) — NOT stored in state, computed inline:**
    - `hourlyLoss = revenue / 30 / 24` (perda por hora)
    - `dropImpact = revenue * 0.20` (impacto queda 20%)
    - `accumulated = hourlyLoss * 5` (5h/mes de downtime)
    - `hasValue = revenue > 0` (gate for conditional rendering)

    **Section structure (per D-01, D-02, 13-UI-SPEC.md Section Structure):**
    ```

            Calculadora de Perdas
          </Badge>
          <h2 className="text-4xl md:text-5xl font-black tracking-tight mb-6">
            Quanto Você Perde{' '}
            <span className="bg-gradient-to-r from-primary to-purple-600 bg-clip-text text-transparent">
              Por Hora?
            </span>
          </h2>
          <p className="text-lg text-muted-foreground max-w-2xl mx-auto">
            Cada minuto de downtime tem um custo real. Descubra o impacto financeiro antes que aconteça.
          </p>
        </div>

        <!-- Calculator container (per D-08) -->
        <div className="bg-destructive/5 border border-destructive/20 rounded-2xl p-8 md:p-12">

          <div className="max-w-md mx-auto mb-8">

              <div className="grid md:grid-cols-2 gap-6 mb-6">
                <Card className="hover:shadow-2xl hover:border-destructive/50 transition-all duration-300">
                  <CardContent className="pt-6">
                    <div className="flex items-center gap-2 mb-2">
                      <TrendingDown className="h-5 w-5 text-destructive" />
                      <p className="text-sm font-medium text-muted-foreground">Perda por hora de downtime</p>
                    </div>
                    <p className="text-4xl font-black text-destructive">{formatBRL(hourlyLoss)}</p>
                  </CardContent>
                </Card>
                <Card className="hover:shadow-2xl hover:border-destructive/50 transition-all duration-300">
                  <CardContent className="pt-6">
                    <div className="flex items-center gap-2 mb-2">
                      <AlertTriangle className="h-5 w-5 text-destructive" />
                      <p className="text-sm font-medium text-muted-foreground">Impacto de queda de 20% nas vendas</p>
                    </div>
                    <p className="text-4xl font-black text-destructive">{formatBRL(dropImpact)}</p>
                  </CardContent>
                </Card>
              </div>

              <!-- Accumulated estimate (per D-07) -->
              <p className="text-center text-sm text-muted-foreground mb-2">
                Com 5h/mês de downtime: <span className="font-semibold text-destructive">{formatBRL(accumulated)}</span>
              </p>

              <!-- Disclaimer (per D-10) -->
              <p className="text-center text-xs text-muted-foreground mb-8">
                Estimativa baseada em médias do setor
              </p>

              <!-- CTA (per D-03) -->
              <div className="text-center">
                <Button asChild size="lg" className="bg-gradient-to-r from-primary to-purple-600 hover:opacity-90 text-white">

                    Começar Grátis — Proteja Suas Vendas
                    <ArrowRight className="ml-2 h-4 w-4" />
                  </Link>
                </Button>
              </div>
            </>
          )}
        </div>
      </div>
    </section>
    ```

    **Important — add `ArrowRight` import** from lucide-react (used in CTA, same pattern as Hero section).

    **Anti-patterns to avoid:**
    - Do NOT use `type="number"` on input — use `type="text"` (per D-04, Pitfall 2)
    - Do NOT use `useState` for timer — use `useRef` (Pitfall 3)
    - Do NOT manually prepend "R$" in formatBRL output — `Intl.NumberFormat` with `style:'currency'` already includes it (Pitfall 4)
    - Do NOT store computed values in state — compute inline from `revenue` (React 18 best practice)
    - Do NOT import any server-only modules (Pitfall 1)
